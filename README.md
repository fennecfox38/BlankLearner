<사용법> BlankLearner.exe와 같은 경로에 File폴더를 만든 뒤 다음 파일들을 폴더 안에 저장해 주세요.
Article.txt에 본문을 붙여넣으세요.
Exception.txt에 절대 빈칸화 시키지 않을 단어를 입력하세요.
VocaList.txt에 무조건 빈칸화 시킬 단어를 입력하세요.(대소문자 구분)
